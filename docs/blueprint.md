# **App Name**: AIthlete

## Core Features:

- Intelligent Program Designer: An AI-powered tool that analyzes user biometrics and fitness history to generate a tailored workout plan and recovery schedule.
- Performance Analytics Suite: A high-fidelity dashboard featuring responsive charts to track calories, weight trends, BMI, and weekly activity volume.
- Integrated Nutrition Log: Full-stack module for managing meal planning, water intake, and nutrient density calculations using a MongoDB backend.
- Workout Engine: Comprehensive workout manager including a real-time timer, daily streak tracking, and exercise categorization.
- Premium SaaS Layer: Monetization module featuring tiered subscription access, badge rewards, and Stripe payment gateway integration.
- Auth & Secure Management: Enterprise-grade authentication system using JWT with user profile management and protected API routes.
- Founder Admin Terminal: An internal interface for monitoring user activity, managing subscriptions, and visualizing platform revenue analytics.

## Style Guidelines:

- Primary color: Dynamic Cobalt (#476BF4) representing energy and high-performance precision.
- Background: Clean Pearl (#F8F9FC), a desaturated variant of the primary hue providing a sophisticated, futuristic backdrop.
- Accent: Sky Cerulean (#3099D1) used sparingly for high-contrast indicators and visual accents.
- Font pairing: 'Space Grotesk' (sans-serif) for high-impact techy headlines, and 'Inter' (sans-serif) for clear, neutral body text.
- Thin-stroke, minimal line icons that complement the futuristic glassmorphism style of the interface.
- Glass-morphic card containers with soft multi-layered shadows, deep rounded corners, and a mobile-first dashboard architecture.
- Smooth micro-interactions for button hovers and staggered fade-in animations for dashboard components to evoke a premium SaaS feel.
